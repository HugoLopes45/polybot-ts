# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Please report vulnerabilities via [GitHub Issues](https://github.com/HugoLopes45/polybot-ts/issues) with the `security` label. Include:

1. A description of the vulnerability
2. Steps to reproduce the issue
3. Potential impact
4. Suggested fix (if any)

## Scope

This policy applies to the `@polybot/sdk` npm package and this repository. Third-party dependencies are out of scope, but we appreciate reports about vulnerable dependencies.
